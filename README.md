## 🚀 Deploy on GitHub Pages

1. Create a new repository on GitHub
2. Push all files to the repository
3. Go to **Settings → Pages**
4. Set Source to **Deploy from branch: main / (root)**
5. Click **Save** — your site will be live within minutes

```bash
git init
git add .
git commit -m "Initial commit: LexCounsel Advocate Website"
git branch -M main
git remote add origin https://github.com/yourusername/lexcounsel.git
git push -u origin main
```
